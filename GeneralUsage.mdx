## General usage of Voice Harbor API

<Steps>
  <Step title="Token Managment.">
    1.1 [Use your admin token to create usage token's.](https://docs.nijta.com/api-reference/admin/developer-token).
  </Step>
  <Step title="Submision Managment.">
    2.1 [Use the usage token to create a job.](https://docs.nijta.com/api-reference/jobs/create)

    2.2 [Select](https://docs.nijta.com/BuildwithVoiceHarbor/Model) and [Define parameters.](https://docs.nijta.com/BuildwithVoiceHarbor/JobParameters#parameters) 

    2.3 [Submit your file(s).](https://docs.nijta.com/api-reference/files/upload-url)
  </Step>
  <Step title="Download protected audio and trascription.">
    3.1 [Use your usage token and job id to download the results.](https://docs.nijta.com/api-reference/files/download-url)
  </Step>
</Steps>
