## Parameters

<ResponseField name="files" type="list" required>
List of local input audio file path(s).  

Example: `['/path/to/file1.mp3', '/path/to/file1.mp3', ...]`
</ResponseField>

<ResponseField name="model" type="string" default="mini">
Model tier to use for phi reduction. Pass `advanced` if you want to use advanced reasoning via our private-LLM. Available model's: advanced, mini.

Example: `job_params = {"files": ['/path/to/file1.mp3', '/path/to/file1.mp3', ...], "model":"mini"}`
</ResponseField>

<ResponseField name="task" type="string" default="protect">
  Task to apply to attached files. Available task's: protect, transcribe.

Example: `job_params = {"files": ['/path/to/file1.mp3', '/path/to/file1.mp3', ...], "task":"transcribe"}`
</ResponseField>

<ResponseField name="agents" type="list" default="health-generic">
  Agents with specififc domain dependency and purpose. Available agents: health-generic, clinical, hipaa.

Example: `job_params = {"files": ['/path/to/file1.mp3', '/path/to/file1.mp3', ...], "model":"advanced", "agents": ["health-generic"]}`
</ResponseField>
